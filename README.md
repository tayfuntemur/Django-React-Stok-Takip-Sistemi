# Stok Yönetim Sistemi

Django + React ile geliştirilmiş stok takip uygulaması.

## Özellikler
- Ürün yönetimi
- Barkod sistemi
- Satış takibi
- Kasa yönetimi
- REST API

## Kurulum

### Backend
\`\`\`bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
\`\`\`

### Frontend
\`\`\`bash
cd frontend
npm install
npm start
\`\`\`
